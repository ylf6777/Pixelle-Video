# Pixelle-Video × zealman 前端集成方案

## 现状

已打通：zealman_client.py + Zealman Test 页面 + G03 图生视频 API

待集成：zealman 能力接入主流程（AI分析 → 编辑分镜 → 生成视频）

## 实施步骤（4步）

### Step 1：修复 zealman_client 独立可用（已完成 ✓）
- zealman_client.py 不依赖 comfykit
- 通过面板 8443 API 直连

### Step 2：zealman 画布地址持久化
在 config.yaml 或 st.session_state 保存 zealman URL
每次启动自动检测可用性

### Step 3：接入主流程的预览/生成按钮
在 style_config.py 的 preview 逻辑中，检测 workflow_source == "zealman" 时：
  1. 上传图片到 zealman `/api/comfy/upload/image`
  2. 提交任务 `/api/workflow/generate`
  3. 轮询结果 `/api/workflow/result`
  4. 返回 media URL

### Step 4：联动分镜编辑器
在 output_preview.py 的生成流程中：
  1. 用户确认分镜内容
  2. 每个分镜的画面提示词 → zealman 图生视频
  3. 拼接所有视频片段

## 接口映射

| Pixelle-Video 概念 | zealman API |
|-------------------|-------------|
| 图片生成 | POST /api/workflow/generate + workflow_id (A01文生图) |
| 视频生成 | POST /api/workflow/generate + G03 workflow_id |
| 上传图片 | POST /api/comfy/upload/image |
| 获取结果 | GET /api/workflow/result?prompt_id= |
| 健康检查 | GET /api/health |
| ComfyUI 管理 | POST /api/comfy/start/stop |

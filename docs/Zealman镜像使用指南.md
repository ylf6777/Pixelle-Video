## 开机流程
1. AutoDL选镜像 → 创建实例
2. 进入控制面板 → 启动ComfyUI
3. 若打不开 → 点WebUI-6006

## Pixelle-Video集成流程
1. ComfyUI GUI中先跑通工作流
2. 导出API格式: Export → API
3. 修改提示词节点为CR输入框（$prompt.text!标记）
4. 放入Pixelle-Video workflows目录
